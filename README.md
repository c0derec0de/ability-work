# Проект AccessJob

Несмотря на развитие удалённой работы, многим людям с ограниченными возможностями по-прежнему сложно найти подходящую вакансию. Существующие сайты поиска работы редко учитывают потребности пользователей с особыми требованиями к доступности.

Идея AccessJob заключается в том, чтобы создать удобную платформу, где люди с ОВЗ смогут искать подходящие заказы, общаться с работодателями и находить работу в комфортной среде.

При разработке особое внимание уделяется доступности интерфейса: крупным элементам управления, высокой контрастности, поддержке скринридеров и удобной навигации с клавиатуры.

## Для доступности реализовано:

- поддержка навигации с клавиатуры;
- семантическая HTML-разметка;
- высокий контраст интерфейса;
- крупные элементы управления;
- поддержка скринридеров;
- возможность увеличения размера текста;
- адаптивный интерфейс для мобильных устройств.

## Основные возможности:

- авторизация пользователя;
- просмотр списка вакансий и заказов;
- поиск по ключевым словам;
- фильтрация вакансий;
- просмотр подробной информации о вакансии;
- отклик на вакансию;
- личные сообщения с работодателем;
- профиль пользователя;
- история откликов.

## Технологии

- React;
- TypeScript;
- Effector;
- React Router;
- Vite;
- CSS.

## Как запустить

Нужен Node.js версии 20.19 или новее.

```bash
npm install
npm run dev
```
